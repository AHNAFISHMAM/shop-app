# Star Café Menu Images

This directory contains images for all Star Café menu items.

## Image Naming Convention

Images should be named according to the dish name in lowercase with hyphens:
- `kacchi-chinigura.webp` → Star Special Kacchi Biryani (Chinigura)
- `beef-masala.webp` → Beef Masala
- `butter-chicken.webp` → Butter Chicken

## Supported Formats

- **.webp** (recommended - best compression)
- **.jpg** / **.jpeg**
- **.png**

## Recommended Image Specifications

- **Resolution**: Minimum 800x600px (landscape or square)
- **Aspect Ratio**: 4:3 or 1:1 preferred
- **File Size**: Under 500KB per image
- **Style**: Professional food photography with good lighting

## How to Add Images

### Option 1: Via Admin Panel (Recommended)
1. Login as admin
2. Go to `/admin/menu-items`
3. Click on a menu item card
4. Click the image area or "Change Image" button
5. Upload image via drag-drop or file selector

### Option 2: Bulk Upload via Admin
1. Login as admin
2. Go to `/admin/menu-items`
3. Click "Bulk Image Upload" button
4. Upload multiple images at once
5. System will auto-match images to dishes by filename
6. Manually assign any unmatched images

### Option 3: Manual File Placement
1. Add your image files to this directory (`public/images/menu/`)
2. Name them according to the convention above
3. Images will automatically be accessible at `/images/menu/your-image.webp`

## Placeholder Images

If no image is uploaded, the system will automatically generate a placeholder with:
- Golden gradient background (#C59D5F)
- Dish name overlay
- Fork & knife icon

## Tips for Best Results

1. **Good Lighting**: Use natural or well-lit photography
2. **Clean Background**: Neutral or restaurant-themed backgrounds
3. **Focus on Food**: Make the dish the main subject
4. **Consistency**: Try to maintain similar style across all images
5. **Appetizing**: Show the dish in its best presentation

## Compression

The admin panel automatically compresses images to optimize web performance while maintaining quality.
