---
description: Read screenshot/image file
keybinding: ctrl+shift+i
---

Help the user read an image or screenshot:

1. Ask for the file path if not provided
2. Check common screenshot locations:
   - Pictures/Screenshots folder
   - Downloads folder
   - Desktop
   - Temp folder
3. List recent image files in those locations
4. Once path is provided, use the Read tool to view the image
5. Analyze and describe what you see in the image

Make it easy by showing them recent .png, .jpg, .jpeg files they might want to show!
